Number Search

Search the catalogue by call number, ISBN, ISSN, publisher number, or catalogue record number.

Call Number

Most items in the library are shelved by LC call numbers.

Try these searches: QA 402, QA 402.6.Y36 2005.

Government Document Call Number

Government documents are located in the Maps, Data, and Government Information Centre (MADGIC).

Try these searches: CA1 BSC 1996 95-253, UN/ST/ECA/171. When searching CODOC numbers, type punctuation exactly as it appears.

Other Call Number

. These include theses, cassettes, compact discs, maps, DVDs, computer games, and selected reserve materials.

Try these searches: GVD (video games), GVB (board games), 1336.H332 (maps), M.M.S 2000 (theses).

ISBN, ISSN, or Publisher Number

Try these searches: 0773500545 (ISBN), 07735 (publisher number).

Record Number

Record number format is a letter followed by 7 numbers: b1000098, e1000541.
